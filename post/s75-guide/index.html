This is an unofficial Chinese version of the Satisfaction 75 build guide. Most of the figures shown in this guide were referred to the official build guide.

1-Satisfaction75 组装

在组装之前，请先对PCB进行测试，刷机，确认其正常工作。如何测试刷机？, 见第2部分内容。

部件清单

  • S75 键盘外壳
  • PCB
  • 尼龙螺柱及螺丝
  • ALPS编码器
  • O rings (O形环)
  • OLED (显示屏)
  • 螺丝
  • 键盘脚垫
  • 导电棉块 (不在图中)

1.1 安装OLED以及导电棉块

首先安装尼龙螺柱如图示位置，以确保显示屏安装在正确的位置:

将尼龙螺柱放置在pcb孔位 (绿圈位置)。

在pcb反面用螺丝将其固定。

将OLED放置在底座上。

焊接OLED引脚，注意将显示屏保持水平。

撕下USB口下方的白色保护膜，将导电棉块贴在标识"Form"的位置，如图所示。

### 1.2 拆卸外壳

  • 拧开背面8颗螺丝。

  • 再拧开上盖的12颗螺丝，取出定位板。

1.3 安装轴和卫星轴

  • 安装卫星轴，如图所示。

  • 盖上定位板。

  • 插入轴体，将其焊接。安装时注意将轴按到底，确保轴体齐平。

1.3 编码器安装

  • 将编码器放入图示位置，焊接。
  • 插线测试轴体触发，编码器触发，及OLED正常工作。

1.4 安装电池 (非必要)

  • Satisfaction75配有一个实时时钟。为了在关机时也正常运行，需另购CR1025型号的纽扣电池。

  • 将电池装进电池座，正极朝上。

1.5 外壳安装

  • Satisfaction75 采用了"burger mount"的结构：将定位板夹在O形环中间。
    将O形环套入M2x6mm六角螺钉，如图所示。

  • 将另外的O形环放置在上盖的螺丝孔位。

  • 拧紧带O形环的六角螺钉，将定位板固定于上盖。

  • 合上底壳，拧紧螺丝。

  • 贴上脚垫。

1.6 安装完毕

  • 装上键帽和旋钮，芜湖，起飞！

2-PCB测试刷机

  • 由于QMK的限制，如果在没有安装OLED的情况下向PCB添加OLED支持，固件将无法工作。
  • 为了能在安装前测试PCB，出厂固件不支持OLED。
  • 您需要按照以下步骤刷机以启用OLED。

如果您已经完成了pcb得测试和刷机，点击此处下载官方固件。

2.1 PCB测试

  • 首先将PCB背面Boot0开关拨到位置"0"，再插线将pcb与电脑连接。

  • 用镊子或其他导电的工具接通每个轴位的轴脚轴座，测试触发。

可用zF的键盘测试工具。

2.2 刷机

一次性设置

  • 安装最新版本的Zadig。(只支持Windows)

  • 下载最新的QMK Toolbox。

  • 将Boot0开关拨到位置"1"

  • 插上线，

  • 打开Zadig，

  • 从菜单栏选择 Options 中的 List all Devices 选项，显示所有设备，

  • 在设备列表下选择 STM32 BOOTLOADER，

  • 在如图所示的驱动列表下选择 WinUSB，

  • 点击 Replace Driver，

  • 你会看到显示 Installing Driver 的进度条。

S75刷机

  • 下载官方固件，点击此处。

  • 打开 QMK Toolbox, 点击"Open"按键。

  • 文件类型选择 .bin，找到并选择刚刚下载好的QMK固件。

  • 点击 Flash 按钮

  • 你将会看到File downloaded successfully表示刷机成功。

  • 将Boot0开关拨回位置"0"，再按下标示"reset"的按钮，完成。

注意：此固件只能在OLED安装上时才能正常使用!
